# `vision_radius` 可遗传：落地、实测与一个被几何天花板决定的负结果

这份文档记录 `docs/trait_roadmap.md` §7.2 点名的"主攻方向"——把感知半径
`vision_radius` 从 `Config` 常量变成每个体的基因——的**完整落地与六配对种子实测**。
路线图把它排在第一位的理由是它同时是 A 类性状（收益从出生持续兑现）**和**
`docs/mortality.md` §1.4 列出的三条渴死瓶颈缓解路线之一（"更远的水感知"），
"做它不是绕开瓶颈，是用性状演化本身去解瓶颈，一步做两件事"（路线图 §4）。

**本文档的核心结论是一个负结果，而且是一个结构性的、由几何天花板预先决定的负结果**：
在水系统修复之后的当前世界里，把 `vision_radius` 变成可遗传基因**没有**缓解渴死
瓶颈，也**没有**朝路线图预期的"向上漂移到中间平衡"方向演化。真正的原因不在代价
项配置，而在 `CLAUDE.md` 记录的那条硬约束本身：**感知格天花板把任何个体的实际
视野封死在 21.33 世界单位，仅比中性起点 21.0 高 0.33 单位，而当前世界食草动物到水
的平均距离是 45.5 单位**。路线图 §1.4 说缺口"只有 4.7 单位"——那是旧世界的数字，
且即便如此，§7.2 自己要求的天花板也只允许视野增长 0.33 单位，远不足以填补缺口。
**路线图 §1.4 的"用视野填补缺口"与 §7.2 的"封顶在感知格以下"两条要求内部互相
矛盾，这份实测把这个矛盾摆到了台面上。**

标记约定同 `docs/biology.md`/`docs/trait_roadmap.md`：`[现实]` 已确立的真实世界
科学；`[本世界实测]` 在这个模拟里跑出来的数字；`[对应]` 原则落到了哪个参数/机制；
`[提案，非结论]` 尚未实现、尚未验证的构想。

---

## 1. 一句话结论

`[本世界实测]` heritable=True vs heritable=False，六配对种子（0–5）× 20000 步：
**两臂的种群平均视野基因无显著差异**（配对 Wilcoxon p=0.156，见 §6），
**渴死占比无显著变化**（True 0.546 vs False 0.542，p=0.84），
食草动物到水距离、渴死年龄均未按预期移动。`carnivore_frac` 与种群规模两臂都在
健康区间，性状没有把生态带崩。**这条性状在当前几何下是近中性的：选择用不上它，
因为它能买到的额外水感知距离（≤0.33 单位）相对于要跨越的水缺口（45.5 单位）
可以忽略不计。**

## 2. 落地了什么（严格照 `trait_roadmap.md` §7.2）

一次 `trait_dim` 变更（3→4，`vision_index = brain_params + 3`），种群作废一次，
`scripts/golden.json` 用 `--bless` 重录（见 §5 与 commit 说明）。逐文件：

- **`underworld/config.py`** `[对应]`：`trait_dim` 3→4；新增 `vision_min=8.0`、
  `vision_span=26.0`（仿 `size_min`/`size_span`：gene=0 → sigmoid 0.5 →
  `8+0.5·26 = 21.0`，等于旧的固定 `vision_radius`，新种群从旧行为起步）；
  `vision_cost=0.02`（能量税系数）；`vision_mutation_sigma=0.02`（同 size/invest
  的慢速率）；消融开关 `vision_heritable: bool = True`；派生属性 `vision_index`
  与 `sense_cell_size = world_size/sense_grid`。
- **`underworld/state.py`** `[对应]`：新增 `vision_radius_of(genome, cfg)`，
  `min(vision_min + vision_span·sigmoid(gene), sense_cell_size)`——**硬裁剪到感知格**，
  就是这条裁剪、而不是 `vision_span`，构成实际上界（见 §3）。
- **`underworld/sensors.py`** `[对应]`：`vision_heritable=True` 时按行读
  `vision_radius_of(state.genome, cfg)`，驱动两处——邻居 `closeness`（第 69 行）
  **和水通道 `water_ch`（第 145 行）**。水通道是这条性状"缓解渴死"的作用点，
  必须一起改，否则基因只影响捕食/同类可见性而与水无关。`False` 臂对全体读固定
  `cfg.vision_radius`。
- **`underworld/dynamics.py` `metabolize`** `[对应]`：加一项
  `vision_cost · max(vision − cfg.vision_radius, 0)` 的**能量**税，**绝不碰
  `thirst` 的水账本**（路线图 §5"解药变毒药"，见 §4）。税与消融开关同门：
  `vision_heritable=False` 时税为零，使 False 臂成为干净的中性对照。
- **`underworld/step.py`** `[对应]`：像 `size` 一样每步从基因组重算 `vision`
  传给 `metabolize`。
- **`underworld/genome.py` `mutate`** `[对应]`：`vision_index` 列以
  `vision_mutation_sigma` 突变。**`crossover` 刻意不豁免 vision**（与 size 不同）：
  把它和永远取自亲本 A 的 diet 基因连锁，会让食肉/食草谱系的视野靠搭便车而非
  选择产生差异，污染 §6 报告的 `carn_vision`/`herb_vision` 对比。
- **`underworld/metrics.py`** `[对应]`：append 四个字段（绝不插入，wire 纪律）——
  `mean_vision`、`vision_std`、`carn_vision`、`herb_vision`，全部报告**裁剪后的
  实际视野**（个体真正用来感知的量）。
- **`tests/test_kernel.py`** `[对应]`：新增 `test_vision_gene_bounded_clipped_and_recombining`
  与 `test_vision_tax_hits_energy_not_water_and_gates_on_ablation`；两个 LOS 遮挡
  测试改 `vision_heritable=False`（它们靠 `vision_radius=40` 当字面感知半径，与
  本性状正交）。全 52 项通过。

`check.py --contracts` 全绿（`in_dim` 未变、`genome_size=1382=brain_params+trait_dim`、
感知格覆盖、协议偏移不动——四个新 Metrics 字段经 `_asdict()` 按名暴露，wire 无需改）。

## 3. 感知格天花板：真正的约束，以及路线图内部的矛盾

`[现实/对应]` `CLAUDE.md` 与 `docs/trait_evolution.md` §22.4 反复记录的硬约束：
**感知格边长必须 ≥ 任意个体的视野半径**，否则超出 3×3 邻域块的个体对视觉和捕食
都静默不可见。当前配置 `world_size=512`、`sense_grid=24`：

```
感知格边长 = 512 / 24 = 21.333 世界单位
中性视野   = 21.0
可增长余量 = 21.333 − 21.0 = 0.333 世界单位   ← 这就是全部头顶空间
```

`vision_radius_of` 因此把实际视野裁剪到 21.333。名义上基因映射到 [8, 34]，但
34 只是让**基因**可以表达"想要更多视野"的意愿；**任何个体的实际感知半径永远
封在 21.333**。这条裁剪，不是 `vision_span`，是真正的上界。

**为什么这让整条性状失效**：水感知的作用距离 = 视野 + `food_sample_dist`（9.0）。

| 量 | 旧世界（`mortality.md` §1.3） | 当前（水修复后，本实测） |
| --- | --- | --- |
| 食草动物到水平均距离 | 34.7 | **45.5**（`herb_water_dist`，六种子均值） |
| 中性水感知距离 | 21+9 = 30 | 30 |
| **天花板下最大水感知距离** | **21.33+9 = 30.33** | **30.33** |
| 缺口 | 34.7 − 30 = 4.7 | **45.5 − 30 = 15.5** |
| 天花板能填补的缺口 | 0.33 | 0.33 |

`[对应]` **路线图 §1.4 说缺口"只有 4.7 单位"，据此把 `vision_radius` 排为主攻；
但同一份路线图 §7.2 要求的天花板只允许视野涨 0.33 单位。0.33 ≪ 4.7 ≪ 15.5。**
两条要求从一开始就互相矛盾：要么放弃天花板（触发已知的静默不可见 bug），要么
接受这条性状能买到的水感知增量在数值上可以忽略。水系统修复把种群推得更向内陆
（种群 +79%，`herb_water_dist` 34.7→45.5），使缺口更大、这条性状更无力。

`[提案，非结论]` 唯一能给这条性状真正头顶空间的办法是**改感知格本身**——增大
`sense_cell_size` 需要*减小* `sense_grid`（更大的格子），但 `k_neighbors=24` 的
每格容量上限会让更大的格子丢弃溢出的邻居；或者把**水通道**（一个地形场采样，
本不受 3×3 邻域块限制）的感知半径与**邻居通道**（受限于感知格）解耦，让水感知
可以超过感知格而邻居感知不超。这两条都改变形状/语义、都要再作废一次种群，且都
超出本次任务范围，留作 `[提案，非结论]`。

## 4. 代价记在能量账本，不记在水账本（路线图 §5）

`[对应]` 税项在 `dynamics.metabolize`（能量），不在 `dynamics.thirst`（水）。
路线图 §5 的"解药变毒药"：一条被提议来缓解渴死的性状，如果把代价记在水轴上，
携带大视野基因的幼体反而耗水更快，主动加重它本该解决的瓶颈。已验证安全的先例是
`carn_cost`——同样"性状值越高持续代价越高"，但记在能量账本，从未表现出体型基因
式的反向选择。税用 `max(vision − baseline, 0)`：低于中性的视野不收费（没有"越瞎
越省水/省能"的白嫖可farm），只对超出中性的部分收费。**但因为实际视野被裁剪在
21.33，可收税区间只有 [21.0, 21.33] 这一条缝，税几乎是惰性的**——这也是为什么
不能把 True 臂的下漂归因于"税太重"（见 §7）。

## 5. 方法

`[对应]` 六配对种子（seed 0–5），每种子两臂 `vision_heritable=True` / `False`，
各 20000 步（`run_headless.py 20000 1000 --json --set vision_heritable=…`）。
GPU 共享，分两波各 6 进程。两臂共享同一基因组布局（trait_dim=4），只差"基因是否
被读/被税"，是干净的 A/B。统计：配对 Wilcoxon signed-rank + 10000 次 bootstrap 的
配对差 95% 区间，报告每个种子的数字，**不做 Bonferroni**，把算过的每个 p 都报出来
（`CLAUDE.md`/`conventions.md` §5）。

**功效诚实说明** `[对应]`（`conventions.md` §5）：种子只变 founder、不变地形，
所有空间结论只推广到*这一套河系*（伪重复）。六种子对 `inland_frac`/`herb_water_dist`
量级的小空间位移**功效不足**——检出 0.02 的 `inland_frac` 移动约需 21 个配对种子。
因此下面任何"未见显著空间效应"都要读作"在六种子功效下未检出"，不是"证明为零"；
但视野基因本身的均值差是个体级量、方差小，六种子足以给出可信区间。golden 因
`trait_dim` 变更必然失配（种群作废），已 `--bless` 重录并在 commit 说明——这是预期，
不是把band放宽掩盖失败。

## 7. 解读：漂变还是选择？

**(a) 视野没有向上演化，两臂都漂到中性以下。** 路线图 §7.2 的可证伪预测是"基因从
中性起点（21.0）向上漂移到某个中间平衡"。实测**证伪**了它：True 臂实际视野均值 19.02、
False 臂 20.16，**都低于 21.0**，方向与预测相反。

这条**下漂的主因是几何裁剪的截断，不是选择**——这正是 A/B 对照要回答的问题。
`vision_radius_of` 把实际视野封在 21.33：基因往上突变超过对应阈值后表型静默饱和
（都映射到 21.33），往下突变则如实表达。于是即便**基因**均值停在中性 0 附近，
被裁剪的**表型**均值 `E[min(8+26σ(g), 21.33)]` 也被拉到 21.0 以下——上侧被削平、
下侧不削。**False 臂（基因既不被读也不被税，纯中性漂变）就已经坐在 20.16，这直接
证明了下漂是裁剪截断的产物，与选择无关。**

**(b) 功能臂比对照臂再低一点，但不显著。** True 比 False 低约 1.13 单位
（bootstrap CI [−2.16, −0.08] 勉强不含零，但配对 **Wilcoxon p=0.156** 不显著；
六种子里 4/6 True<False、2/6 反向）。方向是**向下**，不是路线图预期的向上；即使
这点弱下压是真的，也只说明当视野变得有功能时选择**更**不看重它，而非看重它。
六种子对这个量级的功效不足以下定论，只能说"未检出向上选择，若有方向也是向下"。
`carn_vision` 的 d=−1.45（p=0.063）比 `herb_vision`（p=0.156）稍强，但同样不过线，
不足以支持"食肉谱系视野分化"的任何结论。

**(c) 瓶颈完全没被撼动。** `death_thirst_frac` d=+0.004（p=0.844，CI 含零），渴死
平均年龄 d=+2.4 步（p=0.688），`herb_water_dist` d=−0.60（p=0.313，方向还与"看得远
就敢往更远处走"的预测相反）。三条本该联动下降/上升的指标**全部纹丝不动**。原因在
§3 已经先验给出：天花板下最大水感知距离 30.33，水缺口 45.5，性状能买到的 0.33 单位
增量相对 15.5 单位的缺口可以忽略，选择自然无从用它压低渴死。

**(d) 没有副作用，生态没被带崩。** `carnivore_frac`（0.210 vs 0.206，p=1.0）、
`late_carn`（p=1.0）、`population`（p=1.0）、`min_pop`（1823 vs 1837，无一接近灭绝）
两臂无差异，护栏全部守住。几个次级指标的 bootstrap CI 勉强不含零但 Wilcoxon 不显著
——`carn_water_dist` +2.5（p=0.156）、`mean_water` +0.23（p=0.094）、
`death_starvation_frac` +0.009（p=0.156）——按 §5 的功效说明与"报告每个 p"纪律，
这些一律记为**可疑但功效不足的弱信号，不是结论**。若非要给一个机械叙事：功能臂的
视野略低 →水通道略钝 →稍多个体偏向内陆觅食（`carn_water_dist` 略升）、稍多饿死
（starvation 略升）——但都不过显著线，不应写成因果。

## 9. 结论与后续

**这是一个结构性负结果，且是可以事前从几何推出来的负结果。** 在水系统修复之后，
把 `vision_radius` 变成可遗传基因既没有缓解渴死瓶颈（`death_thirst_frac` 岿然不动），
也没有朝路线图预期的方向演化（视野不升反降，且下降主要是感知格裁剪的截断假象）。
**根因不是代价配置，而是 `CLAUDE.md` 的感知格硬约束与路线图"用视野填水缺口"的目标
从一开始就互相排斥**：天花板只给 0.33 单位头顶空间，而当前世界的水缺口是 45.5 单位
（且水修复把种群推得更向内陆、缺口比旧世界的 34.7 更大）。任务预设的诚实假设——
"如果水修复之后 `vision_radius` 不再显著缓解瓶颈（瓶颈已从 83% 降到 55%），那本身
就是重要发现"——得到证实，而且比"瓶颈变小所以边际收益变小"更强：**即使瓶颈仍在
55%，这条性状在当前几何下根本没有可用的作用空间。**

对路线图的直接反馈 `[对应]`：
- **§1.4 与 §7.2 内部矛盾**，需要在路线图里显式标注：把 `vision_radius` 列为"缺口
  只差 4.7 单位"的主攻手段，与"封顶在感知格以下"（余量 0.33）不能同时成立。
- **Stage 2 的分叉判据**（路线图 §11：`death_thirst_frac` 是否实质下降）在这条
  性状上取**"未下降"分支**——不应据此解锁 B 类性状，而应转向 `mortality.md`
  另两条瓶颈路线，或先动几何。

`[提案，非结论]` 若要让"更远的水感知"这条 `mortality.md` §1.4 路线真正可测，必须
先动几何而非再调基因，三条互斥的方向：
1. **增大感知格**（减小 `sense_grid`）——但 `k_neighbors=24` 每格容量会溢出丢邻居，
   要同时加 `k_neighbors`，两者都是形状参数、再作废一次种群，且更慢。
2. **把水通道与邻居通道的感知半径解耦**——水通道是地形场采样，本不受 3×3 邻域块
   限制，可以让它的半径超过感知格而邻居半径不超；这样视野基因能真正加大水感知
   距离而不触发静默不可见 bug。这是三条里最对症的，但改变 `sensors.sense` 的语义、
   需要单独的证伪实验。
3. **承认 `vision_radius` 不是这个世界的可用杠杆**，把渴死瓶颈的缓解让给
   `mortality.md` 另两条路线（社会学习 / 非 trait 版亲代照料主动向幼体转移水），
   它们不受感知格天花板约束。
